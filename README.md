# ClimateGuard AI

## Project Overview

ClimateGuard AI is a Deep Learning based Climate Forecasting and Risk Analysis System.

The system analyzes historical climate data and predicts future temperature trends using LSTM (Long Short-Term Memory) networks.

## Features

- Climate Forecasting
- Temperature Prediction
- Climate Risk Analysis
- Data Visualization
- Interactive Streamlit Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- TensorFlow
- Keras
- LSTM
- Streamlit
- Scikit-Learn

## Dataset

Global Weather Repository Dataset

## Model

LSTM (Long Short-Term Memory)

## Workflow

Dataset
→ Data Cleaning
→ Feature Selection
→ Scaling
→ Sequence Creation
→ LSTM Training
→ Prediction
→ Risk Analysis
→ Streamlit Dashboard

Dataset:
https://www.kaggle.com/datasets/nelgiriyewithana/global-weather-repository
